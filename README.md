# 每日喝水提醒器 - MyFinal

田敬萱 42311103


功能包括：
1. 查询全国主要城市实时天气（使用和风天气 API） 
2. 根据天气情况，推荐每日合理饮水量
3. 喝水提醒核心功能，手动记录喝水，统计当天喝水量
4. 每天定时发送提醒，如果当天晚8点没有喝到1000ml就发通知

---

## 今日饮水量推荐算法

推荐饮水量计算公式如下：
已知基础饮水量（常温下）：成年人每日大约需要 1500~2000 毫升
推荐饮水量（毫升） ≈ 1500 + 温度 × 20 - 湿度 × 5

## 技术说明

平台：Android Studio
